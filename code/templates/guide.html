An API key is a unique passcode that allows CA Scribe to "talk" to AI models like Gemini or GPT-4o. It acts like a secure handshake between our app and the AI provider.

‚úÖ All keys listed below are 100% FREE to generate!

‚ú® Google Gemini (Recommended)

  1. Go to Google AI Studio.
  2. Sign in with your Google Account.
  3. Click "Create API key".
  4. Select "Create key in new project" (or existing).
  5. Copy the key starting with AIza... and paste it into Scribe.

‚ö° Groq (Fastest)

  1. Go to Groq Console.
  2. Sign in with Google.
  3. Click "Create API Key".
  4. Give it any name (e.g. "Scribe").
  5. Copy the key starting with gsk_... and paste it into Scribe.

üêô GitHub Models (OpenAI GPT-4o)

  1. Go to GitHub Settings > Developer Settings.
  2. Select "Personal access tokens (classic)".
  3. Click "Generate new token (classic)".
  4. Note: You do NOT need any specific scopes/permissions for Models. Just create it.
  5. Alternatively, sign up for GitHub Models Marketplace access if prompted.
  6. Copy the token (starts with ghp_...) and paste it into Scribe.
